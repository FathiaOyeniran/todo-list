About This App

This is a simple, elegant to-do list application built with vanilla HTML, CSS, and JavaScript. It helps you organize your tasks efficiently with features like:

  • Add, edit, and delete tasks
  • Set due dates and times
  • Organize tasks by categories
  • Mark tasks as complete
  • Filter tasks (All, Pending, Completed)
  • Dark mode support
  • Data persistence with localStorage
  • Mobile-responsive design

How to Use

  1. Adding Tasks: Fill in the task title, optional due date/time, and category, then click "Add Task"
  2. Managing Tasks: Use the checkboxes to mark tasks complete, or click the edit/delete icons to modify tasks
  3. Filtering: Use the filter buttons to view All, Pending, or Completed tasks
  4. Dark Mode: Toggle the dark mode switch in the header to change the theme

About the Developer

This application was built as a demonstration of modern web development practices using vanilla JavaScript. It showcases clean code structure, accessibility features, and responsive design principles.

Technologies Used:

  • HTML5 with semantic markup
  • CSS3 with Flexbox and responsive design
  • Vanilla JavaScript (ES6+)
  • localStorage for data persistence
